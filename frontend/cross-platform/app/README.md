# App Directory

Contains all application screens and navigation configurations.

## Structure
*   `index.tsx`: Redirects to home or onboarding.
*   `(auth)/`: Authentication screens (Login, Signup).
*   `(tabs)/`: Main App Tab Bar (Home, Map, Post, Profile).
*   `post/`: Post details and creation screens.
*   `postcards/`: Postcard specific workflows.
*   `profile/`: User details and settings.
*   `onboarding.tsx`: New user intro flow.

## Navigation
Powered by [Expo Router](https://docs.expo.dev/router/introduction/).
