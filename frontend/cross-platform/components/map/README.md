# Map Components

Map visualization and spatial interaction components.

## Platform Support
*   **Mobile**: MapLibre GL Native (High performance vector maps).
*   **Web**: Leaflet (Standard raster maps).

## Key Components
*   **`MapComponent`**: The main map view. Exposes methods like `flyTo`.
*   **`TeleportButton`**: The "Random Mode" trigger button.
*   **`ShutterFlash`**: A white-overlay animation component used during teleportation.
*   **`Filmstrip`**: Horizontal scrollable list of memories visible in the current viewport.
