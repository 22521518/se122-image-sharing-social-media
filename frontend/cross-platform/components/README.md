# Components

Shared UI library for the application.

## Categories
*   `ui/`: Generic, reused elements (Buttons, Inputs).
*   `social/`: Feed, Posts, Comments, Likes.
*   `map/`: Map viewing, markers, teleport interactions.
*   `moderation/`: Reporting and safety tools.
*   `postcards/`: Postcard specific UI.

## Core Themed Components
*   `ThemedText`: Auto-adjusts color based on system theme.
*   `ThemedView`: Auto-adjusts background based on system theme.
