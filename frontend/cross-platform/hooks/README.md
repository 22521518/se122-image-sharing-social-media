# Hooks

Reusable React logic.

| Hook | Description |
|------|-------------|
| `useColorScheme` | Detects current theme (Dark/Light). |
| `useThemeColor` | Helper to resolving a color from props or theme config. |
| `useUserProfile` | Fetches active user data. |
| `useTeleport` | Manages the "Random Memory" animation and state sequence. |
